# AitoVideo — Ограничения и запреты

## Критично (остановка деплоя)

### База данных
- ❌ НЕ удалять таблицу `videos`
- ❌ НЕ менять тип колонки `videoId` (TEXT)
- ❌ НЕ удалять колонки без миграции
- ❌ НЕ менять путь к файлу БД (`backend/data/videos.db`)

### API
- ❌ НЕ менять формат ответа `/api/videos` без версионирования
- ❌ НЕ удалять существующие endpoints
- ❌ НЕ менять авторизацию Mini App

### Telegram
- ❌ НЕ менять команды бота (/start, /help, etc.)
- ❌ НЕ менять формат сообщений бота без согласования

## Требует human review

### Изменения
- Изменение схемы БД (добавление колонок — ок, удаление — review)
- Новые внешние API вызовы
- Изменение деплой-скрипта
- Обновление зависимостей (major versions)

### Добавление
- Новые парсеры видео (проверить легальность)
- Новые команды бота
- Новые переменные окружения

## Разрешено без review

- Исправление багов (если не меняет API/схему)
- Рефакторинг с сохранением поведения
- Добавление тестов
- Обновление документации
- UI изменения в Mini App
- Добавление колонок в БД (nullable по умолчанию)
